# Database

A simple class of PDO to use with any SQL Database

## Install

Via Composer

``` bash
$ composer require cba85/database
```

## Usage

``` php
    use \Database\PDO\SQL as DB;

    $sql = new DB('host', 'db', 'user', 'pass');
```
